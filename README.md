La Página Web

HTML
Una página web consta de tres etiquetas obligatorias: html, head y body
Los elementos de una página web se construyen con etiquetas:
Títulos - h1,h2,h3,...
Párrafos - p
Imágenes - etiqueta sin cierre
Listas ordenadas y desordenadas
